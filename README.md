# Smart-inventory-and-expiry-tracker
Smart Inventory &amp; Expiry Tracker is a Java-based web application for efficient inventory management and expiry monitoring. It uses HTML, CSS, JavaScript, Java HttpServer API, JDBC, and MySQL to add, view, delete products, generate expiry alerts, and reduce product wastage through timely tracking.

# Problem Statement
Managing inventory manually can lead to expired products, stock mismanagement, and financial loss. This project solves the problem by storing inventory data digitally and automatically tracking expiry dates.

# Tech Stack

### Frontend
HTML5
CSS3
JavaScript
Fetch API
### Backend
Core Java
Java HttpServer API
JDBC
### Database
MySQL

# Features

Add new products with validation
Automatic category suggestion
View all products
Delete products
Category-wise expiry alerts
Dashboard analytics
Text-based inventory report generation
Manufacture and expiry date validation


# How to Run the Project

1.Install JDK and MySQL
2.Create the database and products table
3.Update database credentials in DBConnection.java
4.Compile the project
5.Run WebServer.java
6.Open browser and go to: http://localhost:8080
